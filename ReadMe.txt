Language used: Java
Version: Java 9.0.1

Instructions to run:
1. cd to src folder
2. javac Main.java
3. java Main <eta> <lambda> <iterations>

Arguments Info:
eta: learning rate
lambda: regularization parameter
Iterations: hard limit on the number of iterations for gradient ascent

Output:
Accuracy using Multinomial Naive Bayes Without filtering stop words
Accuracy using Multinomial Naive Bayes with filtering stop words
Accuracy using Logistic Regression without filtering stop words
Accuracy using Logistic Regression with filtering stop words
